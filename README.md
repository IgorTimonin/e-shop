# e-shop тестовое задание для Webbooster

## Технологии:
- HTML.
- CSS,
- AJAX,
- React,
- JavaScript

## Установка и запуск приложения:
Клонировать репозиторий:

  git clone https://github.com/ia-stepanov/react-shop.git

Установить зависимости:

  npm install

Запустить приложение:

  npm run start
